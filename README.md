# spring-boot-mybatis-kylin

#### 项目介绍
一个Demo项目：
基于SpringBoot集成Mybatis访问Kylin
基于StringBoot集成Mybatis访问麒麟

#### 软件架构
    SpringBoot
    Mybatis
    Kylin-JDBC
    Tomcat-JDBC(连接池)


#### 安装教程

    mvn clean package
    java -jar KylinApplication.jar

#### 使用说明

    curl 'http://127.0.0.1:8080/account/all'

#### 迭代计划

    demo项目、除了BUG，无迭代计划
